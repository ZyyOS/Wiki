项目简介

Zyy Maple Leaf OS 是一个动画精美、UI 现代化的 Scratch 伪操作系统。它旨在通过 Scratch 的图形化编程能力，模拟真实操作系统的交互逻辑，提供极致的编程体验和视觉享受。

本项目不仅仅是一个游戏，更是一个教育平台，展示了图形化编程在复杂逻辑构建上的可能性。

核心特性

  • 纯粹体验： 去除冗余设计，还原最纯粹的操作系统交互逻辑，窗口管理、文件系统等一应俱全。
  • 极速响应： 优化了Scratch的运行效率，无需Scratch即可独立运行，提供接近原生应用程序的流畅度。
  • 安全沙盒： 创新的权限管理系统，每个应用在隔离环境中运行，保护核心系统安全。
  • 多端支持： 支持 x86 和 ARM 架构，可在 Windows、MacOS、教育平板等多种设备上运行。

安装与部署

你可以通过以下方式获取 Zyy Maple Leaf OS：

  1. 访问 官方网站 下载最新版本 v1.2 (26N16-snapshot-3测试版)。
  2. 解压文件后，直接运行主程序。
  3. 如果你是开发者，请联系 ZYY-Office 参与开发。

开发指南

我们欢迎所有开发者参与贡献。项目主要使用 基于 02Engine (Scratch)进行开发，部分底层逻辑可能涉及 JavaScript 扩展。

详细的 API 文档和贡献指南，请参阅 贡献名单页面。

常见问题

Q: 这个系统可以在手机上运行吗？
A: 目前开发主要倾向于PC端，移动端适配正在开发中。

Q: 是否开源？
A: 目前暂不开源，但未来有计划开放部分核心代码。
